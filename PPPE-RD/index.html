Développeur web

Mon père travaillant dans l'informatique, je suis depuis tout petit dans ce domaine. J'ai d'ailleurs créé mon premier site vers mes 14 ans ce qui m'a permis de découvrir l'envers du décors des sites web et j'ai adoré cela. Dans ce site je vais donc parler du métier de développeur web, un métier que je souhaiterais exercer. Nous allons donc traiter la problématique suivante : qu'est-ce que le métier de développeur web ? Afin de réaliser mes recherches, j'ai utiliser comme mots clefs : développeur web, back-end, front-ent, full-stack.

Conclusion

Nous avons pu voir à travers les différentes références ci-dessus que le métier de développeur web ne consiste pas seulement à coder un site web. En effet le développeur web possède différentes tâches qui va de faire le style du site à rendre les boutons fonctionnels par exemple. Il doit également travailler en équipe avec les différents corps de métier qui permettent de faire un site web. J'ai mis 7h pour faire cette page web (recherche et conception de la page. Afin de mener à bien cete recherche, j'ai utilisé le moteur de recherche Google et Youtube. Les recherches effectués m'ont permis d'en découvrir un peu plus sur le métier de développeur web et d'apprendre à citer des sources.
